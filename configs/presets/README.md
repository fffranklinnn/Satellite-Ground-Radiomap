# Presets

Scene-oriented config presets.

- `xian_urban.yaml`: Xi'an urban core, `urban_flat`
- `qinling_mountain.yaml`: Qinling area, `mountain_rural`
- `huashan_mountain.yaml`: Huashan area, `mountain_rural`
- `loess_plateau.yaml`: Loess Plateau area, `mountain_rural`

Adjust `time.*` and `layers.l1_macro.tle_file` for the target date before running.
