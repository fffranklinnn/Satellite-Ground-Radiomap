# L1 Space Layer Data

This directory contains data for the L1 Macro/Space layer.

## Required Data

- **TLE files**: Satellite orbit data
- **Antenna patterns**: Satellite antenna gain patterns
- **Weather data**: Atmospheric parameters (optional for V2.0)

## Example TLE File

Place your satellite TLE file here, e.g., `satellite.tle`

## Example Antenna Pattern

CSV format with columns: elevation, azimuth, gain_dbi
