# L2 Terrain Layer Data

This directory contains Digital Elevation Model (DEM) data for terrain analysis.

## Required Data

- **DEM files**: .tif (GeoTIFF) or .hgt (SRTM) format
- Coverage: Should cover the simulation area

## Data Sources

- SRTM: https://www2.jpl.nasa.gov/srtm/
- ASTER GDEM: https://asterweb.jpl.nasa.gov/gdem.asp
- USGS Earth Explorer: https://earthexplorer.usgs.gov/

## Example

Place DEM files here, e.g., `N39E116.hgt` for Beijing area
